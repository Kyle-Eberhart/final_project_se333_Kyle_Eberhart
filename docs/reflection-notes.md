# Reflection

This project demonstrates an AI-assisted testing workflow using a software agent.

The agent analyzes application code and generates unit tests to improve coverage.

The iterative cycle involves:
- generating tests
- running the build
- analyzing coverage results
- generating additional tests

The Spring Petclinic application provided a realistic target for testing.

One challenge was constraining the agent to only test specific classes to avoid generating an excessive number of tests.

This project demonstrates how AI tools can automate portions of software testing while maintaining transparency through version control.
